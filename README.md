# User Tools для Telegraf

Модуль для ботов на Telegraf, который добавляет три команды для работы с Telegram Bot API:

- `/user` — краткая информация о пользователе  
- `/avatar` — все доступные аватарки пользователя  
- `/debug_user` — полный JSON-объект

Команды работают:

- без аргументов - с самим отправителем команды  
- с `id` — по числовому Telegram ID  
- с `@username` — по username пользователя

---

## Возможности

- Получение имени и `id` пользователя или чата:
  - `/user`
  - `/user 1234`
  - `/user @username`
- Просмотр всех доступных аватарок пользователя:
  - `/avatar`
  - `/avatar 1234`
  - `/avatar @username`
- Получение полного объекта:
  - `/debug_user`
  - `/debug_user 1234`
  - `/debug_user @username`

---

## Ограничения Telegram

> Telegram Bot API не позволяет боту получать данные о пользователе,  
> если этот пользователь **никогда не взаимодействовал с ботом**  
> (не писал ему `/start`).

В таких случаях команды вернут сообщение:

```text
Этот пользователь не взаимодействовал ранее с ботом
```

## Добавление в проект
1. Клонирование / добавление файла

- `Добавьте файл модуля в свой проект, например:`

```text
project/
  bot.js
  get_commands.js
```
2. Добавьте в ваш основной файл проекта
```text
const getCommands = require("./get_commands.js");
getCommands(bot);
```
